## Setup

1. Copy all your artwork images into the `images/` folder with the exact filenames above.
2. Open `index.html` in any browser — no build step needed.
3. Works entirely with plain HTML, CSS, and vanilla JS.

---

## Image Tips

- **artist.png** — Portrait/headshot of Divya. Best in portrait orientation (3:4 ratio).
- **art1.png–art9.png** — Artworks for the gallery grid. Portrait (3:4) ratio looks best.
- All images are cropped to fill their containers; top-aligned for portraits.

## Customization

- Colours and fonts are defined as CSS variables at the top of `css/style.css`
- Gold accent: `#c9a84c`
- To change fonts, update the Google Fonts link in `<head>` and the `--font-serif` / `--font-sans` variables
